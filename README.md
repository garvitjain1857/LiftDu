# LiftDu

An alternative cab service app to other apps like Ola, and Uber, which control the work environment for drivers and take chunks of cuts from their earnings as commission, making life difficult for a regular driver. LiftDu will connect drivers directly with customers to share a ride. The transactions will go through smart contracts run on blockchain technology and paid via cryptocurrency. Anyone can register as a driver and publish rides for people to book. This DAPP makes the process more hassle-free, making it more transparent.

## Set-Up

1. Install Yarn
```
npm install --global yarn
```

2. Install Dependencies:
```
yarn install
```

## To Run on Localhost:
1. 
```
cd LiftDu
```
2. 
```
yarn run dev
```

### Other Commands

1. Compile Contracts:
```
yarn hardhat compile
```

2. Run Scripts:
```
yarn hardhat run <script name>
```
3. Run tests:
```
yarn hardhat test
```
